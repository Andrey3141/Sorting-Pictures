# CHANGELOG.md

## [1.2.0] - 2026-04-02

### Added
- Умная сортировка фото по папкам с именами людей
- Система обучения модели: добавление новых лиц через UI
- Механизм "ложных срабатываний" - возможность помечать области как НЕ ЛИЦО
- Пакетная обработка целых папок с фото
- Защита от дубликатов через MD5 хеширование файлов
- Автоматическая пересортировка фото после дообучения модели
- Drag-and-drop загрузка фото в интерфейс
- Профессиональный UI с анимациями (PyQt6)
- Детальное логирование всех операций в реальном времени
- Прогресс-бар для пакетной обработки

### Changed
- Улучшена точность распознавания через FaceNet эмбеддинги
- Оптимизирована работа с базой данных SQLite
- Улучшена обработка ошибок при загрузке фото

### Fixed
- Исправлена проблема с дублированием фото при повторной сортировке
- Исправлена ошибка сериализации JSON при сохранении ложных срабатываний
- Исправлен баг с определением лиц на фото с несколькими людьми

- Диалоговое окно для обучения модели

## [1.0.0] - 2026-04-02

### Added
- Базовое распознавание лиц через FaceNet
- Детекция лиц через Haar Cascade
- Сохранение неизвестных лиц в базу данных
- Сортировка фото по известным/неизвестным лицам

### Security
- Хеширование файлов для предотвращения дубликатов
- Локальное хранение данных без отправки в облако
